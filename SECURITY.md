# Security Policy

## Reporting

Please report security or privacy issues privately through GitHub's private vulnerability reporting if it is enabled for the repository. If it is not available, open a minimal issue asking for a private contact path without including exploit details.

## Scope

Security-sensitive areas include:

- Unexpected network requests
- Clipboard access
- Local stream history storage
- Background audio behavior
- Malicious or malformed media URL handling

## Privacy Principles

M3U8r should remain free of analytics, tracking, advertising SDKs, and server-side collection. Any proposed change that stores, transmits, or derives user data should be discussed before implementation.
